What about the capital letter W?

Technically, a capital letter W repeated 63 times is the longest pixel length combination of 63 characters.


MMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMM.INTERNATIONAL

WWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWWW.INTERNATIONAL


The only problem with this argument is that domain names are not case-sensitive. The only characters of a URL that could potentially be case-sensitive are the characters that come after the domain name (the /path); this is entirely dependent on the server and its environment. By default, browsers automatically make domain names lower-case.


Because of this technical rule, one could make the argument that the pixel length of a domain name is solely based on it lower-case form, thus making m*63.international the longest domain name.
